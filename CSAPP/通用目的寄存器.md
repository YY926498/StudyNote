# 通用目的寄存器

x86-64的中央处理单元（CPU)包含一组16个存储64位值的通用目的寄存器。这些寄存器用来存储整数数据和指针。

名字及用途分别如下：

- `%rax`	返回值
- `%rbx`	被调用者保存
- `%rcx`	第4个参数
- `%rdx`	第3个参数
- `%rsi`	第2个参数
- `%rdi`	第1个参数
- `%rbp`	被调用者保存
- `%rsp`	栈指针
- `%r8`	第5个参数
- `%r9`	第6个参数
- `%r10`	调用者保存
- `%r11`	调用者保存
- `%r12`	被调用者保存
- `%r13`	被调用者保存
- `%r14`	被调用者保存
- `%r15`	被调用者保存

规则：生成1字节和2字节数字的指令会保持剩下的字节不变；生成4字节数字的指令会把高位4个字节置为0
